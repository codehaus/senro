Provides a Service Provider Interface for diverse metadata sources. AOP is used by MetadataFactory to provide multiple inheritance functionality against the classes returned by a MetadataProvider. The MetadataManager uses the factory to create the Metadata classes that are returned.

Related Documentation

  • The Java Metadata Interface, JSR-40, provides a solid foundation of how universal metadata can be implemented in Java. This implementation is strongly based on the work and ideas that were developed there.
  • JSR-40 is based on OMG MOF.
  • The JSR-40 reference implementation is the Netbeans Metadata Repository
